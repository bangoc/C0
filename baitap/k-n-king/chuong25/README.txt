1. Viết chương trình kiểm tra xem tùy chỉnh địa điểm bản địa của trình biên dịch của bạn có phải là địa điểm "C" hay không?

2. Viết chương trình nhận tên địa điểm từ tham số dòng lệnh và sau đó hiển thị các giá trị được lưu trong cấu trúc lconv cho địa điểm đó. Ví dụ, nếu địa điểm là "vi_VN" (Việt Nam), thì đầu ra của chương trình có thể như sau:
decimal_point = ","
thousands_sep = "."
grouping = "\3"
mon_decimal_point = ","
mon_thousands_sep = "."
mon_grouping = "\3"
positive_sign = ""
negative_sign = "-"
currency_symbol = "₫"
frac_digits = 0
p_cs_precedes = 0
n_cs_precedes = 1
p_sep_by_space = 0
n_sep_by_space = 0
p_sign_posn = 1
n_sign_posn = 1
int_curr_symbol = "VND "
int_frac_digits = 0
int_p_cs_precedes = 0
int_n_cs_precedes = 1
int_p_sep_by_space = 0
int_n_sep_by_space = 0
int_p_sign_posn = 1
int_n_sign_posn = 1
Để dễ đọc, các ký tự trong grouping và mon_grouping cần được hiển thị dưới dạng mã số.
